# Changelog

All notable changes to the "BuySmartlyNow" project will be documented in this file.

## [2.1.0] - 2025-12-18
### Added
-   **Price Comparison List**: Restored the multi-retailer list in product cards, allowing users to compare prices across different platforms (Jumia, Konga, Amazon) directly.
-   **Best Deal Highlight**: The lowest price is visually highlighted as the "Best Deal".
-   **Click Tracking**: Added logic to simulate tracking clicks on individual retailer links.

## [2.0.0] - 2025-12-18
### Changed
-   **Visual Identity Overhaul**: Implemented "Deep Blue & Orange" design system.
-   **Typography**: Switched to `Poppins` (headings) and `Inter` (body).
-   **Navbar**: Converted to a sticky layout with integrated search bar.
-   **Footer**: Redesigned with 4-column layout and trust badges.
-   **Hero Section**: Implemented auto-rotating banners with "Compare. Save. Earn." messaging.
-   **Product Cards**: Updated design with "Hot Deal" badges and visual savings calculation.
-   **New Sections**: Added "Categories" icon grid and "Earn With Us" affiliate promotion section.
